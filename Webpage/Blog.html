    • Semantic markup

    • What is semantic markup? Semantic markup is using HTML in such a way as to make your webpage more meaningful. Semantic HTML is not visible to the user when accessing the webpage as CSS determines the styling, but it is very important to non-visual users. You can avoid using semantic HTML, but your webpage will not have any meaning to those users. Semantic HTML should not be used for styling, only to convey meaning.

    • What is accessibility and why is it important?

    • What is accessibility? Accessibility refers to the ability to access various places, services, etc. by everyone, especially for people with disabilities. Examples of accessibility in public are: elevators/ramps for people with physical and mobility impairments, braille and beeping from traffic lights for blind people, etc. When referring to accessibility online, we are referring to the ability to access web content by people with disabilities, who make use of things such as screen readers or subtitles.
