Style guide

    • The first image is the first draft of my homepage, with the intention of using a similar layout for all pages. I initially planned on using a button to access the menu, but decided against it in later versions. I also planned on putting my about information on the homepage, but later put it on its own page. The second image is the menu that would be shown when clicking on the menu button. A similar menu will be shown when clicking on the menu button in mobile layouts.
    • The horizontal image is the current layout for the PC version of my homepage, and the vertical image is the current layout for the mobile version of my homepage. I wanted to keep the page simple, but include features that people expect when viewing most webpages. At the top of the page is the header with the menu, which remains at the top of the screen even when you scroll down the page. The footer remains at the very bottom of the webpage, so it only becomes visible once you have scrolled all the way down. The content of the homepage includes the title of my webpage, and icons which are used to navigate to other pages.
    • These images show the current layout for the PC and mobile versions of my about page. The page features the title of the page, an avatar representing myself, and a description of me.
    • These images show the current layout for the blog homepage. From there you will be able to select an article to read. The links will consist of the title and an excerpt, either of which may be clicked.
    • These images show the current layout of the blog article pages. There will be a back button to go to the blog homepage at the top of the article. The page will consist of the blog title and blog content.
  • The homepage for my work, and the actual game pages will use a similar layout to the blog pages. These pages will also feature images or video clips from my work.
